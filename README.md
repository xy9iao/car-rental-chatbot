# Car Rental Chatbot Spring Boot

This project is a Java Spring Boot web application that extends a Prompt Engineering workshop sample into a simple AI-powered car rental chatbot.

## Features

- Spring Boot REST API
- Simple web chat UI
- Car rental assistant prompt
- Grounded responses using a car rental dataset
- AI API integration

## Tech Stack

- Java 17
- Spring Boot
- Maven
- HTML/CSS/JavaScript
- OpenAI/OpenRouter API

## Project Flow

User message → Spring Boot Controller → Chat Service → Car Dataset Search → Prompt Builder → AI API → Chatbot Reply
